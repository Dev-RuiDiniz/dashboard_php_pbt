# Skill local do Codex

Este repositório referencia um skill local do Codex para manutencao e extensao da aplicacao:

- Nome do skill: `maintain-dashboard-php-pbt`
- Caminho local instalado: `C:\Users\RUI FRANCISCO\.codex\skills\maintain-dashboard-php-pbt`
- Arquivo principal: `C:\Users\RUI FRANCISCO\.codex\skills\maintain-dashboard-php-pbt\SKILL.md`

## Objetivo

O skill foi criado para orientar tarefas recorrentes neste projeto, com foco em:

- fluxo da aplicacao PHP MVC customizada
- rotas em `config/routes.php`
- RBAC e permissões em `App\Services\AuthorizationService`
- alteracoes de schema, migrations e `database/final_mvp.sql`
- validacoes locais antes de release ou deploy Hostinger

## Como invocar

Use prompts como:

```text
Use $maintain-dashboard-php-pbt to add a new admin CRUD screen.
```

```text
Use $maintain-dashboard-php-pbt to debug a route, permission, or SQL issue in this repo.
```

## Observacao

O skill nao fica versionado dentro do repositorio. Este documento existe apenas para apontar para a instalacao local e registrar a convencao de uso.
